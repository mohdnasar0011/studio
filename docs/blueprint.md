# **App Name**: FitConnect

## Core Features:

- Community Feed: Display a feed of local workout posts and questions, similar to a Reddit feed, showing user avatars, reliability scores, text content, optional images, and upvote/downvote buttons.
- User Geolocation: Determine the user's current geographical location to enable the Nearby feed toggle to accurately determine the list of available users.
- Matchmaking Cards: Implement a card-swiping interface (Hinge-style) that enables users to find workout partners, displaying a large profile photo, name, age, distance, workout goals, preferred time, and pace/level.
- Real-Time Chat: Provide a real-time chat interface (WhatsApp-style) for matched users, enabling them to communicate directly within the app.
- User Reliability Scoring: Uses AI as a tool, evaluate the overall reliability of other app users when displaying each available match.
- Bottom Navigation Bar: Implement a fixed bottom navigation bar for easy access to the Feed, Match, and Chat sections.
